SinglyRequest Class Reference

Inherits from NSMutableURLRequest
Declared in SinglyRequest.h

Overview

SinglyRequest is a subclass of NSMutableURLRequest provided to aid developers in making requests against the Singly API. Requests are initialized with the relevant credentials for the current SinglySession shared instance automatically.

Configuring the Request

  •   endpoint

    The relative endpoint path to access on the Singly API. This path will be appended to the full request URL.

         property
  •   parameters

    The request parameters to use for the request. These will be merged with the access_token and any other required parameters of the Singly API.

         property
  •   isAuthorizedRequest

    Whether or not the request requires authorization. This defaults to YES, however, it can be set to NO to prevent the passing of credentials (which may be necessary in case you have an invalidated access token or for requests that do not need authorization).

         property
